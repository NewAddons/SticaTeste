Política de Privacidade

Exemplo Política de Privacidade da Trans Stica Transportes
1. INTRODUÇÃO

A Trans Stica Transportes, inscrita no CNPJ sob o n° 04.098.837/0001-76, (“Trans Stica” ou “Empresa”), tem o compromisso com a privacidade e a segurança dos seus dados pessoais. Esta Política de Privacidade (“Política”) estabelece como coletamos, usamos, compartilhamos e armazenamos seus dados pessoais, bem como os seus direitos como titular dos dados.

2. DADOS PESSOAIS COLETADOS

A Trans Stica coleta dados pessoais de diversas formas, como:

  • Dados fornecidos por você: nome completo, CPF, RG, endereço, telefone, e-mail, dados bancários, informações sobre a carga a ser transportada, entre outros.
  • Dados coletados automaticamente: informações sobre o seu dispositivo, como endereço IP, tipo de navegador, sistema operacional, data e hora de acesso ao nosso site ou aplicativo.
  • Dados de terceiros: informações obtidas de empresas parceiras, como transportadoras, seguradoras e órgãos governamentais.
3. FINALIDADE DO USO DOS DADOS PESSOAIS

Os dados pessoais coletados pela Trans Stica são utilizados para diversas finalidades, como:

  • Prestar os serviços de transporte de cargas.
  • Efetuar cotações e orçamentos.
  • Emitir documentos fiscais.
  • Rastrear a carga.
  • Manter contato com você durante o processo de transporte.
  • Cumprir obrigações legais, entre outros.
4. COMPARTILHAMENTO DE DADOS PESSOAIS

A Trans Stica poderá compartilhar seus dados pessoais com as seguintes categorias de empresas:

  • Empresas parceiras: transportadoras, seguradoras, órgãos governamentais, empresas de tecnologia, etc.
  • Prestadores de serviços: empresas que nos auxiliam na prestação dos nossos serviços, como empresas de marketing, cobrança e contabilidade.
  • Órgãos governamentais: quando necessário para o cumprimento de obrigações legais.
5. SEGURANÇA DOS DADOS PESSOAIS

A Trans Stica adota medidas de segurança técnicas e administrativas para proteger os seus dados pessoais contra acessos não autorizados, uso indevido, divulgação, perda ou alteração.

6. ARMAZENAMENTO DOS DADOS PESSOAIS

Os seus dados pessoais serão armazenados pelo período necessário para o cumprimento das finalidades descritas nesta Política, ou enquanto você não solicitar a sua exclusão.

7. SEUS DIREITOS COMO TITULAR DOS DADOS

Como titular dos dados, você possui os seguintes direitos:

  • Acessar seus dados pessoais: saber quais dados seus estão armazenados pela Trans Stica e como eles estão sendo utilizados;
  • Corrigir seus dados pessoais: solicitar a correção de dados incorretos ou incompletos;
  • Excluir seus dados pessoais: solicitar a exclusão dos seus dados pessoais, quando possível;
  • Solicitar a anonimização ou o bloqueio de seus dados pessoais: solicitar que a Trans Stica torne seus dados anônimos ou bloqueie seu uso, quando possível;
  • Revogar o seu consentimento: solicitar a revogação do seu consentimento para o tratamento dos seus dados pessoais, quando o tratamento for baseado no seu consentimento;
  • Opor-se ao tratamento dos seus dados pessoais: solicitar que a Trans Stica deixe de tratar seus dados pessoais, em determinadas situações;
  • Portar seus dados pessoais: solicitar que a Trans Stica forneça seus dados pessoais em um formato estruturado, de uso corrente e de leitura automática, para que você possa transferi-los para outro controlador;
  • Reclamar para a Autoridade Nacional de Proteção de Dados: caso você entenda que seus dados pessoais estão sendo tratados de forma irregular, você poderá reclamar para a Autoridade Nacional de Proteção de Dados.
8. CONTATO

Para exercer seus direitos como titular dos dados ou para qualquer outra dúvida sobre esta Política, você pode entrar em contato com a Trans Stica.
